# AEGIS TNST: Soberanía Cognitiva 2026

Repositorio oficial para la publicación del Manifiesto y Whitepaper Técnico de **Soberanía Cognitiva 2026**.

## Visualización

El documento está disponible en línea a través de GitHub Pages:

- **Español (Principal):** [Ver Manifiesto](https://rocuts.github.io/cognitive-sovereignty/)
- **English:** [Read Manifesto](https://rocuts.github.io/cognitive-sovereignty/manifesto_2026_en.html)

## Estructura del Proyecto

- `index.html`: Documento principal (Manifiesto en Español).
- `manifesto_2026_en.html`: Versión en Inglés.
- `assets/`: Recursos estáticos (imágenes, scripts, estilos).

## Despliegue

Este repositorio está optimizado para **GitHub Pages**. Para desplegar:

1. Ir a **Settings** > **Pages**.
2. Seleccionar la rama principal (`main` o `master`) como fuente.
3. Guardar. El sitio estará disponible en minutos.
